# expensia-frontend

Frontend for Expensia, built with Next.js App Router, React, TypeScript, Tailwind CSS,
TanStack Query, Zustand, and Base UI/shadcn-style primitives.

## Run

```bash
yarn install
yarn dev
```

The app proxies `/api/*` requests to `EXPENSIA_API_BASE_URL`, defaulting to
`http://localhost:8080`.
